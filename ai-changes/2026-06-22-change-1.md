## Risks

- The `App.start()` method contains an infinite placeholder loop; running `python -m src.main` without `KeyboardInterrupt` will hang. This is expected for Phase 1 and will be replaced by the real event loop in Phase 2.
- Cross-platform hotkey/audio libraries are not yet integrated, so this phase only validates the skeleton.
- `.env`/environment variable prefix is implicit (pydantic-settings default). Future phases may add a custom env prefix if needed.
