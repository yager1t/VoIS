# AI Change Log — 2026-06-22-change-21

## Reason
Implement Phase 4 of v0.3: an adaptive vocabulary learning system that records
explicit user corrections and extracts recurring candidate terms from dictated
text, promoting them to the user dictionary once frequency thresholds are met.

- `tests/unit/test_app.py` — tests that learning is called/skipped and that

## Risk
- `VocabularyLearner` writes to `data/vocab/corrections.jsonl` and
  `data/vocab/term_counts.json`. A malformed file is handled gracefully by
  resetting counts.
- Learning is disabled by default (`dictionary_learning_enabled=False`), so
  normal dictation behavior is unchanged.
- Term-extraction heuristics may occasionally promote sentence-cased common
  words; the common-word filter and length threshold mitigate this.
